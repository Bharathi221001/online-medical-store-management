# Online Medical Store Management System

A web-based application developed to manage an online medical store, allowing users to browse medicines and manage cart flow.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- User signup page
- Product listing
- Cart management
- Dashboard page
